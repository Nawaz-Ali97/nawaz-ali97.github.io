Python Quiz

This is the first successful Python project which i created. It consists of a general knowledge quiz based on components which make up a PC. At the end the program will tally the correct answers and wrong answers. It will then give a percentage score based on the results.

This mainly consists of IF and ELIF statements in order to determine wether the answer is correct and add to the tally of correct and incorrect scores. The full code to this project can be found on my github profile.

Number Guesser Game

This is the Random Number Guesser game which i created. The user types in a number which will be the upper boundry of the guess. The computer will then pick a random number between 0 and the user input. The user will then have to guess the number with the computer giving hints such as higher or lower. At the end the program will tally the number of user guesses.

This consists of if statements, else statements, elif statements and a while loop. The full code to this is available on my github.

Rock Paper Scissors

This is a Rock Paper Scissors game which i have created in python. The user is asked to select rock, paper, scissors or to quit. Then computer chooses between rock, paper or scissors randomly. The two results are compared in order to determine the winner and a tally of user wins, cpu wins and draws is kept. When the user quits the totals of all the tallys are printed.

This consists of if statements, elif statements and else statements in order to compare the 2 picks. It also uses arrays in order to assign a value of 0 to 2 for rock, paper and scissors. Then using the random command to select an integer between 0 and 2 and storing the value of that integer in the array as the cpu's pick. It also uses a while loop to keep the program running until the user inputs "q" causing the program to print the results and then quit. The full code to this program can be found on my github profile.

Blackjack

Here i have created a blackjack game with python. The user is dealt 2 cards and the dealer is dealt 1. The user can then choose to hit or stay. The game ends when the user stays and the dealer either busts or gets 17 or more. It also ends if either the user or dealer busts. The program than states wether the user or dealer wins and the total of both hands.

This consists of a function to deal cards, calculate the total and reveal the dealers hand. It also uses arrays to store the cards in a deck and face cards. It uses the random command to select a random card from the deck array and then remove it to the array. It the uses a for loop and a while loop in order to go through the multiple circumstances which would cause the game to end. The full code to this program can be found on my github.
